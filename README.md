# Esercizi del corso di programmazione avanzata con TypeScript

## Setup

```sh
npm i -g typescript@latest ts-node@latest
git clone https://github.com/gcanti/typescript-course.git
cd typescript-course
npm install
```

## Come eseguire i test

```sh
npm test -- src/<file>
```
